# XML Uploader

This repository contains a simple Flask application that lets you upload an XML file and saves it to the server.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   python app.py
   ```

3. Open your browser and go to http://127.0.0.1:5000 to upload an XML file. Uploaded files will be saved in the `uploads` directory.
